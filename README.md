**Лабораторная работа №6: Классы, интерфейсы, кеширование**

**Цель работы:**
Освоить работу с классами, перегрузкой операторов, интерфейсами, клонированием и кешированием в C#.

Структура проекта:
**1. Коты (Cat, FireCat, CountMeow, IMeow)**
1. Cat — класс кота с именем.
- Методы Meow() (один раз) и Meow(int n) (N раз).
- Статический метод MakeAllMeow() для вызова мяуканья у коллекции котов.

2. IMeow — интерфейс с методом meow().

3. FireCat — огненный кот с особым мяуканьем.

4. CountMeow — обёртка для подсчёта вызовов meow() (задание 3).

**2.  Дроби (Fraction, CachedFraction, IFraction)**

1. Fraction — класс дроби с числителем/знаменателем.
- Перегрузка операторов +, -, *, /.
- Реализация ICloneable для клонирования.
- Метод Equals и GetHashCode для сравнения.

2. IFraction — интерфейс с методами:
- GetDoubleValue() — получение десятичного значения.
- SetNumerator(), SetDenominator() — установка значений.

3. CachedFraction — кеширующая обёртка:
- Хранит вычисленное значение double в поле _cachedValue.
- При изменении дроби кеш сбрасывается (InvalidateCache()).

**3. XML-документация**

Для всех публичных классов, методов, свойств написана XML-документация с использованием тегов:
- <summary> — краткое описание.
- <param> — описание параметра.
- <returns> — описание возвращаемого значения.
- <example> — пример использования.
- <remarks> — дополнительные пояснения.
